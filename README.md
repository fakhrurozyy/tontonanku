# Tontonanku — Dummy Web

Tontonanku adalah dummy web untuk konsep personal dan sosial watchbook: film, series, anime, rating, review, memory photo, playlist mood, profile user, dan add friend.

## Fitur Dummy

- Landing page editorial minimal
- Mock login modal
- Explore database dummy film/series/anime
- Filter berdasarkan tipe konten
- Profile user dengan tab review, watchlist, friends
- Add friend dummy interaction
- Watch memory photo section
- Responsive mobile-first, siap untuk GitHub Pages

## Cara pakai lokal

Buka `index.html` langsung di browser.

## Upload ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload semua file: `index.html`, `style.css`, `script.js`, dan `README.md`.
3. Masuk ke Settings > Pages.
4. Pada Branch, pilih `main` dan folder `/root`.
5. Save, lalu tunggu link GitHub Pages aktif.
